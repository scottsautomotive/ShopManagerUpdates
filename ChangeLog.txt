Update: The "Diagnostic & Skill Routing" Patch (v2.1)

📊 Active Shop Board Expansion (Kanban)
New "In Diagnostic" Lane: Upgraded the Active Shop Board from 4 lanes to a full 5-lane Kanban workflow. A dedicated purple "In Diagnostic" column now sits between Estimates and In Progress.

Streamlined Routing: Vehicles in the diagnostic phase are now visually separated from vehicles currently undergoing wrench-time (In Progress), giving service writers a more accurate view of shop capacity.

🛠️ Technician Workflow & Security
Diagnostic Tech Notes: The Work Order Editor now intelligently recognizes the "In Diagnostic" status (Status 5) and keeps the Technician Notes text box unlocked, allowing mechanics to log their findings before the job officially moves to "In Progress."

Dashboard Sync: Updated the Technician Dashboard query engine. Cars flagged as "In Diagnostic" now properly route to the technician's Active Jobs grid alongside standard active repairs.

Skill-Based Job Routing: The Work Order Editor now strictly filters the "Assigned Tech" dropdown based on the A/B/C skill level required for that specific labor line. Service writers can no longer accidentally assign Master-level diagnostics to a C-Tech.

📋 Labor Catalog & Inventory
Catalog Skill Assignment: The "Add/Edit Labor" window now includes a RequiredSkillLevel dropdown (A-Tech, B-Tech, C-Tech, or None), mapping permanent requirements to standard shop jobs.

Skill Transparency: Added a new purple "Skill Req." column directly to the Labor Catalog DataGrid so managers can verify job constraints at a glance.

Smart Catalog Search: The Labor Catalog search bar has been upgraded to scan by Skill Level. Typing "A-Tech" will instantly filter the catalog to show only master-level procedures.

🚀 Auto Shop Pro - Release Notes
Update: The "Shop Floor & Payroll" Expansion (v2.0)

🛠️ The Technician Workflow
Labor Line Assignments: Shifted from assigning entire Work Orders to assigning individual labor lines. An A-Tech and a C-Tech can now clock different jobs on the exact same ticket.

The "Tech Blindfold" (Role-Based Security): Technicians have a completely locked-down view. When logged in, they can no longer see shop financials, parts costs, labor rates, tax totals, or front-office tabs (Customers, Appointments, Accounting). They see only the vehicles in their bay and the work they need to perform.

Technician Dashboard: Added a dedicated, gamified dashboard for technicians. Techs can instantly view their active assignments, completed jobs, and live "Flagged Hours" for the current week.

Printable Tech Sheets: Added a dedicated print button for clean, stripped-down physical work orders for mechanics to take to the bay.

⏱️ HR & Master Payroll
Expanded HR Profiles: The User database now tracks advanced employee metrics, including SkillLevel (A, B, C, None), PayType (Hourly, Flat Rate, Salary), StandardHourlyRate, and FlatRateHourly.

Universal Time Clock: Added a persistent "Clock In / Clock Out" toggle directly to the main sidebar, allowing hourly and salary employees to track physical time in the building.

Master Payroll Report: Built a comprehensive back-office dashboard that automatically calculates total shop payroll. It pulls physical time punches for hourly staff, cross-references completed Work Orders for Flat Rate technicians, and generates precise gross pay calculations based on individual HR profiles.

📋 Intake & Front Desk
NHTSA VIN Decoding: Integrated a live REST API connection to the US Department of Transportation. Service writers can now click "Decode" to automatically pull Year, Make, Model, and Engine data from any 17-digit VIN.

Active Shop Board Expansion: Added an orange "On Hold / Waiting on Parts" lane to the Kanban board. Technicians can now digitally flag a ticket as stalled, instantly routing it to the front desk.

Vehicle Quirks & Notes: Added a persistent notes field to the Vehicle Profile for tracking specific customer warnings (e.g., "Driver door sticks," "Needs synthetic oil").

✨ Quality of Life Improvements
Live Data Refresh: The Active Shop Board and Work Order List now silently auto-refresh in the background every 15 seconds, keeping the front desk and the shop floor perfectly in sync without locking up the UI.

Smooth Scrolling: Fixed a global UI bug where hovering over a DataGrid or TextBox would trap the user's mouse wheel and prevent the main window from scrolling.

Check-In Mileage Trap: The Work Order editor now dynamically prompts the service writer to enter the check-in mileage the very first time they open a newly created ticket.

"Split-Brain" Prevention: Added strict Network Guards to prevent laptops from silently creating fake local databases on their C: drive if they temporarily lose Wi-Fi connection to the master PC.

Zombie Record Prevention: Fixed an issue where clicking a Work Order that was recently deleted by another computer resulted in a dead click.

🚀 Auto Shop Manager Pro - Changelog v1.1.0 Release Notes 3/15/2026

🎨 UI & Navigation Upgrades
Streamlined Startup Flow: The application now immediately boots into the Active Shop Board instead of the analytics Dashboard, getting service writers right into the action the second the app opens.

Sidebar Reorganization: The sidebar menu has been optimized for a live shop workflow. The Active Shop Board has been pinned to the very top for priority access.

Analytics Tab: Renamed the "Dashboard" to Analytics and repositioned it directly above the Accounting tab to logically group your financial and reporting tools together.

💻 Network & Multi-Workstation Support
Local Area Network (LAN) Ready: The software can now be deployed across multiple computers in the shop. By dropping a simple db_network_config.txt file into the app data folder, extra workstations will automatically route their connection to a shared master database.

Hardware License Decoupling: Rearchitected the licensing system. License keys are now stored locally on each physical machine rather than inside the shared database, allowing multiple activated PCs to read from the exact same shop data without overriding each other's activation keys.

Database Collision Armor: Upgraded the SQLite connection pipeline with Cache=Shared and a 15-second busy timeout. If the Front Desk and the Shop Floor hit "Save" at the exact same millisecond, the software now intelligently forms a queue to prevent "Database Locked" crashes.

📝 Work Order & Data Integrity
Smart Mileage Prompts: When generating a brand new Repair Order, the editor now creates a dynamic pop-up asking the user to confirm or update the vehicle's incoming mileage, automatically highlighting the text for rapid data entry.

Finalize Validation (Mileage Out): Added strict guardrails to the invoice finalization process. The system now actively checks the "Mileage Out" field. If it is blank, a pop-up forces the user to enter the departing mileage and will safely block the finalization process until a valid number is provided.

⏱️ Labor Catalog
Quick Rate Overrides: Redesigned the Add/Edit Labor Rate window. The Hourly Rate box has been streamlined and now features a dedicated Pencil "Edit" button. Clicking the button instantly unlocks the field, changes the styling, and highlights the text so you can quickly override the shop's global default rate for specific specialized jobs.
